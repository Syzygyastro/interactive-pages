Connection Application Data

What a developer has to submit when applying for a grid connection in Great Britain — separated by transmission level (large generation/storage/demand connecting directly to the high-voltage network) and distribution level (smaller projects connecting via a DNO).
CUSC · Grid Code · Gate 2 G99 · G98 · DCode · DCUSA Sources: GB Industry Codes & Engineering Recs
Transmission-level connection applies to large generation, storage, or demand connecting directly to the GB transmission system. Governed by the Connection and Use of System Code (CUSC), the Grid Code, and (post-reform) the Gate 2 Criteria Methodology. Submission goes via NESO (the system operator) to the relevant Transmission Owner — National Grid Electricity Transmission, SSEN Transmission, or SP Transmission.

Notes

  • Post-TMO4+ reform, the Gate 2 filter is the dominant change: applicants must now evidence land rights and planning maturity before securing a queue position. The pre-reform queue of ~739 GW was stripped to ~283 GW of generation/storage + ~99 GW of demand at Gate 2 in December 2025.
  • The Connections Network Design Methodology (CNDM, Dec 2025) sets how NESO converts a portfolio of Gate 2 applications into a coordinated network design.

Distribution-level connection applies to smaller generation (broadly under ~50 MW) and demand connecting via a Distribution Network Operator (DNO) — the company that owns the lower-voltage local network. Governed by the Distribution Code (DCode), Engineering Recommendation G99 for generation (and G98 for the smallest microgenerators), and DCUSA for the commercial side. The pathway and form depend on the project's G99 Type: A is the smallest (≤1 MW), B is 1–10 MW, C is 10–50 MW, D is ≥50 MW or transmission-connected.

Notes

  • The G98 / G99 split is a size cut-off: G98 covers fit-and-inform installations up to 16 A per phase (typical domestic rooftop solar). G99 covers everything larger.
  • Type A applications are deliberately streamlined — the network impact of a single small generator is local. Larger Types (B / C / D) carry progressively more documentation and stage-gated commissioning, because their fault contribution and voltage impact reach further.
